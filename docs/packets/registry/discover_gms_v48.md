## Serverbound ‚Äî method

harvest** (ctor `??0COutPacket@@QAE@J@Z` @ `0x57b77e`): every code xref to the

delta-doc-flagged CHANGE_CHANNEL/MOVE_PLAYER/WHISPER were **BODY-verified** from
