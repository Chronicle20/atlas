## Serverbound — method

Read from the v48 `COutPacket(N)` send-sites via a **global COutPacket-ctor
harvest** (ctor `??0COutPacket@@QAE@J@Z` @ `0x57b77e`): every code xref to the
ctor was walked, the nearest preceding `push imm` taken as the opcode, and each
caller function name recorded. **317 xrefs → 128 distinct opcodes.** Opcodes were
**never region-shift-predicted** — each is the literal COutPacket argument. The
delta-doc-flagged CHANGE_CHANNEL/MOVE_PLAYER/WHISPER were **BODY-verified** from
their send-sites (decompiled), not inferred from adjacency.
