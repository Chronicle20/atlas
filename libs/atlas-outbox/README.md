# atlas-outbox

## Overview

Transactional outbox library for Atlas services. Provides a reusable building block for atomically persisting domain state changes alongside outbound Kafka events, then relaying those events asynchronously with at-least-once delivery semantics.

This package is a stub. Public API, relayer, and persistence wiring land in later tasks of the dynamic-service-config initiative.
